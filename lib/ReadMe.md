# Required assemblies

Copy the following assemblies from your Content Manager ServiceAPI installation into this folder before building:

- HP.HPTRIM.SDK.dll
- HP.HPTRIM.ServiceAPI.Common.dll
- HP.HPTRIM.ServiceAPI.Model.dll
- ServiceStack.dll
- ServiceStack.Interfaces.dll
- ServiceStack.Text.dll

Depending on your installed CM version, the service assembly may be named differently. If your environment uses older names, update the project references to match the assemblies shipped with that version.
